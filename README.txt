Block Ctrl+Alt+Delete

This program provides functionality to block the Ctrl+Alt+Delete key combination in Windows operating systems.

Usage:
1. Save the "block_ctrl_alt_delete.cpp" file in this folder.
2. Run the "run_on_startup.bat" file to compile and execute the program.
3. Ctrl+Alt+Delete will be blocked every time the computer restarts.

Note:
- Running this program will disable access to the Task Manager.
- Exercise caution when using this feature as it can cause issues with system administration.
